# Week 3 Challenges

## Dicee Challenge

A simple dice game where players roll virtual dice to determine a winner. 

- Open `dicee/dicee.html` in your browser to play.
- All code and images are in the `dicee/` folder.

## Simon Game

A beginner-friendly Simon Game with overlays, practice rounds, and modern UI.

- Open `:simon/index.html` in your browser to play.
- All code, sounds, and instructions are in the `:simon/` folder.

For more details, see the individual README files in each project folder.
